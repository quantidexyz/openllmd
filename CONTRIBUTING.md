# Contributing to `@quantidexyz/openllmd`

This repository is a **read-only mirror**, regenerated from the OpenLLM
monorepo on every release. Do not expect merges here to stick — the mirror's
`main` is force-updated from upstream each release.

**PRs are welcome and are ingested upstream.** A maintainer pulls your change
into the monorepo (your commit + authorship preserved), it goes through review
there, and the next release re-mirrors it back. Your PR is then closed as
"landed upstream in vX".
